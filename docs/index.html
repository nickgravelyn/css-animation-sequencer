Seashell

A simple library for CSS animation wrangling.

Available on Github. Created by Nick Gravelyn.

View source to see code for each example. Documentation coming soon.

Example animations from Animate.css.

Seashell helps manage CSS animations in a Sequence for more interesting animations through composition of separate animations.
Sequences can create relationships between separate DOM elements. When one animation completes, the other begins.

Sequences can repeat once, forever, or any number of times. They can be stopped at any point and can be reused once constructed.

Seashell also provides an AnimationBuilder which makes it easy to dynamically build keyframe based CSS animations, for when you can't define animations up front.
