A lovely library for web animations.

The heart of Seashell is a Timeline that coordinates a series of events. Events can be any object that implements a simple API and Seashell comes included with premade events for CSS animations, delays, style changes, callbacks, running other timelines, and running a number of events concurrently!

Here we're using Timeline to sequence some animations from Animate.css on this happy green box.

While Timeline can support any event matching the right API to support 3rd party events, it does expose a slightly simpler API for built-in events to reduce typing when making timelines. Here's that same animation using this method. All arguments here are simply forwarded to the matching event constructors, so no functionality is lost.

So far this is all well and good, but Seashell goes even further in enabling you to create nice animated sequences, starting with the DynamicCssAnimation class. This utility allows JavaScript to define an animation in code that is then built into a CSS keyframes based animation which can be used by a Timeline.

Timelines can play other timelines which allows for flexible composition of animations. By default the main timeline waits for the child timeline to finish, but the play method can take in options (whether for timeline or predefined CSS animations) to configure settings, such as whether or not to run the timeline or animation asynchronously.
