    public createCssString(): String source

    Creates the CSS for this animation.

    This method is used internally by generateStyle, which is the recommended method to use when using this class for animations. However if you would rather manage inserting the CSS into the document you can call this method to simply get the CSS as a string.

    Return:

    String

    The CSS for the animation.

    public destroyStyle() source

    Removes the style block inserted by generateStyle.

    public generateStyle() source

    Generates the CSS style and appends it to the document head.

    Generally you should call this method before using the animation so the browser has the CSS. Alternatively you can use createCssString to create the CSS and then insert it into the document yourself.

    If you use this method for an animation you don't plan to re-use, you should at some point call destroyStyle to remove the generated style block from the document.
