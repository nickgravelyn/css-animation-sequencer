CSS Animation Sequencer

Manage CSS animations in a simple, small library.

Available on Github. Created by Nick Gravelyn.

Example animations from Animate.css.

CSS Animation Sequencer helps manage CSS animations in a sequence for more interesting animations through composition of separate animations.
Sequences can create relationships between separate DOM elements. When one animation completes, the other begins.

Sequences can repeat once, forever, or any number of times. They can be stopped at any point and can be reused once constructed.
